# Integration test fixtures

このディレクトリはローカル専用の fixture 置き場です。実顧客データを含むため、
`.gitignore` で **どんなファイルも git に追加されない設定** になっています。

## 期待されるファイル

統合テストは以下のファイル名を期待します（無ければ該当テストは skip）:

| ファイル名 | 用途 |
| --- | --- |
| `月額変更_70歳以上含む.zip` | ルート直置き 7140001/7210001 の複数名統合 |
| `資格取得_70歳以上.zip`     | フォルダ構造 + 7100001/7180001/DataRoot 混在 |
| `育児_社保雇保混在.zip`     | [社保]育休（XML）+ [雇保]育児系（PDF同梱）の混在 |
| `扶養.zip`                  | [社保]被扶養者異動 (7170003) + DataRoot 形式併存 |

## ローカルでテストを動かす

1. 上記の ZIP をこのディレクトリにコピー
2. プロジェクト直下で `npm run test:integration` を実行

fixture が存在しないテストは自動的に skip されます。
